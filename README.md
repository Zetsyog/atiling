# ATiling

Perform source to source algebraic tiling optimization

## Requirements

- Pluto (link)
- Trahrhe (link)

## Build

Configure

```
./get_submodules.sh
./autogen.sh
./configure
```

Now run 
```
make
```
to start compiling and 
```
sudo make install
```
to install as a library (WIP)
