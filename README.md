# ATiling

Perform source to source algebraic tiling optimization

## Requirements

- Pluto (link)
- Trahrhe (link)

## Build

Configure

```
./get_submodules.sh
./autogen.sh
./configure
```

Now run 
```
make
```
to start compiling and 
```
make install
```
(often need to sudo) to install atiling as a library
